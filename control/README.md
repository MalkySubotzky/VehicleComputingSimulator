**Installations:**
<br>
```bash

sudo apt-get install libbson-dev
sudo apt-get install nlohmann-json3-dev
sudo apt install pkg-config

```

google tests:
<br>
```bash

sudo apt update
sudo apt install cmake libgtest-dev
cd /usr/src/googletest/googletest
sudo mkdir build
cd build
sudo cmake ..
sudo make
sudo make install

```
